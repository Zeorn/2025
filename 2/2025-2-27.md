# 一、国际综合数据库
 
```
1.OpenSanctions(通缉数据集和制裁数据收集)
OpenSanctions 是一个国际数据库，聚焦具有政治、刑事或经济利益的个人和公司。它整合了制裁名单、涉政敏感人员数据库等多种信息，形成一个易于使用的数据集。利用该数据库，可交叉核对信息，排查利益冲突和非法活动迹象，在国际交易中筛选潜在客户和合作伙伴，跟踪政治冲突，对比各国制裁政策，还能将相关信息融入现有数据产品。
官网：https://www.opensanctions.org/datasets/


2.斯德哥尔摩国际和平研究所（SIPRI）
SIPRI 总部位于斯德哥尔摩，是独立研究冲突、军备等领域的机构。其维护的 5 个数据库，即多边和平行动、军事开支、武器转让、武器禁运和国家报告数据库，数据基于开源信息和年度调查问卷，可下载获取。
官网：https://www.sipri.org/databases
```

# 二、武器相关数据库

```
3.小武器调查组织
该组织位于日内瓦国际与发展研究学院，提供武装暴力互动地图和图表，有可下载数据库。它还链接各类国际报告，网站的透明度晴雨表可评估武器数据真实性，另有识别武器和弹药的工具，如弹药追踪套件等。
官网：https://www.smallarmssurvey.org/databases


4.冲突军备研究组织（iTRACE 冲突武器数据库）
这是欧盟资助的数据库，提供从冲突地区缴获的流失武器和弹药的精确信息，包括制造商细节、图像和卫星地图定位。报告来源明确，包含实地研究，使用需免费注册。
官网：https://itrace.conflictarm.com
```

# 三、冲突研究与数据平台

```
5.奥斯陆国际和平研究所（PRIO）
PRIO 位于奥斯陆，有许多活跃项目和专家。其网站提供 1946 - 2018 年武装冲突和武器数据的可下载数据集，有助于研究冲突趋势。
官网：https://www.prio.org/data

6.挪威小武器转让倡议（NISAT）
NISAT 数据库主要提供小武器、轻武器及其零部件和配件的转让信息，数据多来源于官方政府数据，如海关记录。涵盖 1962 - 2015 年 250 个国家和地区间的数百万条武器贸易记录。
官网：http://nisat.prio.org/Trade-Database/

7.联合国裁军事务厅（联合国报告军事开支数据库）
自 1981 年起，联合国成员国在此自行报告军事开支情况，为研究全球军事开支提供基础数据。
官网：http://www.un.org/disarmament/convarms/Milex/


8.联合国裁军事务厅（联合国常规武器登记册）
该登记册提供各国每年主要武器系统进出口的自报信息，页面链接可访问其他常规武器数据库，如小武器、爆炸物等。
官网：https://www.un.org/disarmament/convarms/register/


9.美国国务院（世界军事开支与武器转让数据库）
提供 170 个国家的年度军事开支、武器转让、武装部队人员等统计数据，数据可追溯至 1964 年，还有分析和重点内容，最新报告于 2021 年发布。
官网：https://www.state.gov/world-military-expenditures-and-arms-transfers/


10.世界银行（军事开支占 GDP 百分比数据库）
通过世界银行的世界发展指标（WDI）数据库，可按国家和时间查询军事开支占比、维和人员数量、武器进出口等数据，其中武器贸易数据来源于 SIPRI。
官网：https://data.worldbank.org/indicator/MS.MIL.XPND.GD.ZS?fbclid=IwAR2cNYKUdJovQu9_1c6Rzqfs9EIFSspCGklGcERMblViMmXRsTjQufCLgD8


11.国际冲突研究所（INCORE）
由北爱尔兰阿尔斯特大学管理，专注于发布对近期出版的种族冲突相关书籍、期刊文章和研究论文的同行评审。它具备强大的搜索功能，支持按主题、年份和作者进行检索，能为特定冲突的研究提供珍贵的档案信息，同时也是挖掘专家资源的重要渠道。不过，其数据库信息的时间跨度仅为 1994 - 2009 年，在使用时需要留意这一局限性 。
官网：https://www.ulster.ac.uk/incore
```

# 四、冲突分析与信息机构

```
12.国际危机组织
总部位于布鲁塞尔的非营利组织，按地区和国家编写冲突报告和分析，网站是获取信息和专家资源的便捷渠道。
官网：http://www.crisisgroup.org/


13.战争相关因素项目（COW）
由宾夕法尼亚州立大学和加州大学戴维斯分校监督，提供经过学术审核的战争、冲突、联盟等数据集，数据时效性因数据集而异。
官网：https://correlatesofwar.org/data-sets/


14.海德堡国际冲突研究所（冲突晴雨表）
德国的冲突研究机构，每年发布量化报告衡量冲突强度，提供武器、人员等信息。网站有多种可下载数据集，工作人员名单也是地区专家资源。
官网：https://hiik.de/?lang=en
```

# 五、政策与援助监测平台

```
15.国际政策中心（安全援助监测）
是华盛顿特区国际政策中心的项目，提供 2000 年以来美国全球安全和国防数据，数据可按国家、地区等多种方式搜索，涵盖军事、警察援助等多个方面。
官网：https://securityassistance.org/

16.欧盟委员会（全球冲突风险指数）
基于开源量化指标，对各国结构性条件评分，预测未来 1 - 4 年暴力冲突风险。指数涵盖 6 个维度 22 个变量，使用 1991 年以来历史数据训练模型，涉及多种冲突类型。
官网：https://drmkc.jrc.ec.europa.eu/initiatives-services/global-conflict-risk-index#documents/1059/list
```

# 六、其他相关平台

```
17.美国国务院（全球刑事司法办公室）
处理战争罪、危害人类罪等问题，提供讲话、报告、政策文件等多种资源。
官网：https://www.state.gov/bureaus-offices/under-secretary-for-civilian-security-democracy-and-human-rights/office-of-global-criminal-justice/


18.欧洲防务局（出版物和数据）
每年收集 27 个成员国国防部的防务数据，在年度报告中公布汇总数据，提供防务支出、投资等关键指标，2006 年起有完整数据集。
官网：https://eda.europa.eu/publications-and-data?fbclid=IwAR1RLUHkP-aX2kmYL0Us_Iub7FULyH-P_swDSzjbp6iwztiOXvbnb3dL0sI


19.欧盟（欧洲防务工业发展计划 EDIDP）
旨在支持欧盟防务工业竞争力和创新能力，网站有相关情况说明书和项目信息。
官网：https://defence-industry-space.ec.europa.eu/eu-defence-industry/european-defence-industrial-development-programme-edidp_en?fbclid=IwAR2I07U3twC1CsqiESJfHhlldqSjwr8p1b6BYlqBb54WqbA4GXaJV4aQhcE


20.欧盟（欧洲防务基金 EDF）
2021 - 2027 年有近 80 亿欧元资金，用于防务合作研究和能力开发项目，网站介绍了资金分配和承包商信息。
官网：https://defence-industry-space.ec.europa.eu/eu-defence-industry/european-defence-fund-edf_en?fbclid=IwAR0ZJSRkOuJJT2A69cZBDNQCQ_si4QHjS7HiMYR39TGY5x5bZF5z5YNnvf4


21.永久结构性合作（PESCO）
欧盟整合防务产业的倡议，项目涵盖培训、陆地、海洋等多领域，网站有各项目信息。
官网：https://www.pesco.europa.eu/?fbclid=IwAR0BEP1Y1J_YUE6u8Hbxak62ST-uD1B8Mr_SjuyJ8BkVb3bZJscuBHULN3s

22.澳大利亚战略政策研究所（国防开支公共数据库）
收集 2001 年以来国防预算和开支数据库，数据可下载，帮助研究人员分析国防事务。
官网：https://www.aspi.org.au/cost-of-defence-database?fbclid=IwAR3S6PU4fP7jJLLBRKNH5rpdOQMWCBGH1Nykp1oWMKMqSzjnBOpCSP6a1GU


23.北约（国防开支信息数据库）
自 1963 年起，北约每年发布成员国财政、人员和经济数据汇编，包括国防开支增长率、占 GDP 百分比等多种数据。
官网：https://www.nato.int/cps/en/natohq/topics_49198.htm?fbclid=IwAR1ZNDzaRVinzOLwWLU4bkkiZ1yVLdbZ1-KBJD4qqa091V5vd4zTLLwdK5w


24.哥伦比亚大学法学院人权研究所（安全部队监测）
致力于提高安全部队透明度和问责制，有美国 2001 年以来对外军事培训和援助数据库，为研究人员提供支持。
官网：https://securityforcemonitor.org/data/?fbclid=IwAR1-21g7QQl8VysMhQDVSI5EJotErZqy2JoBZrWx0Xas1H-jhXMn_ezVwb0


25.法国国家统计与经济研究所（国防工业企业调查）
法国政府每三年公布一次国防工业数据，最近一次更新在 2018 年。
官网：https://www.insee.fr/en/metadonnees/source/serie/s2055?fbclid=IwAR17zTBbLgqn8dpUszcmA7Mtu4dL3RpWTUE4ON6gNZjBv_MIdT70XdHSPK8


26.欧盟统计局（政府国防开支统计）
收集并公布欧盟各国国防开支数据。
官网：https://ec.europa.eu/eurostat/statistics-explained/index.php?title=Government_expenditure_on_defence&fbclid=IwAR3G2USGF1nQ1DEreC8oeUOq3kWXaqtSWhphlbbPkTadm0hLMsb0VQziuqc#:~:text=Expenditure%20on%20'defence'-,In%202020%2C%20'defence'%20expenditure%20amounted%20to%201.3%20%25%20of,(1.6%20%25%20of%20GDP)


27.新美国基金会（无人机世界）
收集开源信息制作全球无人机进出口图表，总结各国和非国家行为体无人机能力，2020 年最后更新。
官网：http://securitydata.newamerica.net/world-drones.html

28.国际刑事法院（文化遗产政策报告）
报告定义了国际武装冲突中针对文化遗产的犯罪，2021 年更新。
官网：https://www.icc-cpi.int/sites/default/files/itemsDocuments/20210614-otp-policy-cultural-heritage-eng.pdf


29.极端主义研究项目（国会山骚乱数据库）
创建与 2021 年 1 月 6 日事件相关的法庭记录数据库，随案件进展更新。
官网：https://extremism.gwu.edu/Capitol-Hill-Siege
```
